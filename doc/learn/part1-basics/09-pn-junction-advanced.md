# PN 结进阶：暗电流与耗尽区分析

从泊松方程到暗电流四大机制：深入 PN 结的定量分析

> 本章为进阶内容，需要先掌握 [PN 结基础](04-pn-junction-basics.md) 和 [半导体基本方程](08-semiconductor-equations.md)。

## 耗尽区宽度与电容

### 泊松方程求解

耗尽区内载流子近似为零（耗尽近似），泊松方程简化为：

$$\frac{d^2\phi}{dx^2} = -\frac{\rho}{\varepsilon_s}$$

其中 N 侧 $\rho = qN_D$，P 侧 $\rho = -qN_A$。

对单边突变结（如 $N_D \gg N_A$ 的 P⁺N 结），耗尽区主要延伸到轻掺杂侧：

$$W \approx \sqrt{\frac{2\varepsilon_s(V_{bi} - V)}{qN_D}}$$

### 耗尽区宽度

总耗尽区宽度为两侧宽度之和：

$$W = x_n + x_p = \sqrt{\frac{2\varepsilon_s}{q}\left(\frac{1}{N_A} + \frac{1}{N_D}\right)(V_{bi} - V)}$$

其中 $V$ 为外加偏置电压（正向为正，反向为负）。反向偏置时 $|V|$ 增大，$W$ 增大。

### 结电容

耗尽区类似平行板电容器，单位面积结电容为：

$$C_j = \frac{\varepsilon_s}{W} = \sqrt{\frac{q\varepsilon_s N_A N_D}{2(N_A + N_D)(V_{bi} - V)}}$$

反向偏置下 $C_j$ 随 $|V|$ 增大而减小——这是变容二极管（Varactor）和 PIN 光电二极管提高响应速度的原理。

## 暗电流

**暗电流** 是光电探测器在无光照条件下流过的电流，是限制探测器灵敏度的核心噪声源。暗电流并非单一机制，而是多种物理过程的叠加。

### 暗电流的组成

#### 1. 扩散电流（Diffusion Current）

少子在耗尽区边缘的热扩散产生。这是理想 PN 结的反向饱和电流 $I_S$：

$$I_{diff} = qA\left(\frac{D_n}{L_n}n_{p0} + \frac{D_p}{L_p}p_{n0}\right)$$

其中 $n_{p0}$ 和 $p_{n0}$ 为少子平衡浓度，$D$ 为扩散系数，$L$ 为扩散长度，$A$ 为结面积。

利用 $n_{p0} = n_i^2/N_A$ 和 $p_{n0} = n_i^2/N_D$：

$$I_{diff} \propto n_i^2 \propto T^3 \exp\left(-\frac{E_g}{k_B T}\right)$$

**特征**：与 $n_i^2$ 成正比，对温度极其敏感。窄带隙材料（如 InGaAs）的扩散电流远大于宽带隙材料（如 Si）。

#### 2. 产生-复合电流（Generation-Recombination Current）

耗尽区内陷阱中心（缺陷、杂质）通过 Shockley-Read-Hall（SRH）过程产生电子-空穴对，被电场扫出形成电流：

$$I_{gr} = \frac{qA n_i W}{\tau_{eff}}$$

其中 $W$ 为耗尽区宽度，$\tau_{eff}$ 为耗尽区有效载流子寿命。

$$I_{gr} \propto n_i \propto T^{3/2} \exp\left(-\frac{E_g}{2k_B T}\right)$$

**特征**：与 $n_i$ 成正比（而非 $n_i^2$），温度依赖性弱于扩散电流。在低偏压和小带隙器件中常占主导。反向偏置增大使 $W$ 增大，$I_{gr}$ 随之增大。

#### 3. 隧穿电流（Tunneling Current）

高掺杂或高电场下，价带电子直接量子隧穿到导带（带间隧穿），或通过陷阱辅助隧穿（TAT）：

$$I_{tun} \propto \exp\left(-\frac{4\sqrt{2m^*E_g}}{3\hbar qE}\right)$$

**特征**：与电场强度强相关，高掺杂结（结宽很窄）中显著。在 InGaAs 等窄带隙探测器的高反向偏置下尤为重要。温度依赖性弱。

#### 4. 表面漏电流（Surface Leakage Current）

器件表面的界面态、氧化层电荷和工艺缺陷导致的漏电通路：

$$I_{surf} \propto \frac{q n_i s A_{peripheral}}{2}$$

其中 $s$ 为表面复合速度，$A_{peripheral}$ 为结周长。

**特征**：与器件制造工艺和钝化质量密切相关。可通过优化台面刻蚀、侧壁钝化等工艺降低。

### 暗电流的温度依赖性

不同暗电流分量对温度的依赖程度不同：

| 暗电流分量 | 温度依赖 | 激活能 |
|-----------|---------|--------|
| 扩散电流 | $\propto T^3 e^{-E_g/kT}$ | $E_g$ |
| 产生-复合电流 | $\propto T^{3/2} e^{-E_g/2kT}$ | $E_g/2$ |
| 隧穿电流 | 弱温度依赖 | ≈ 0 |
| 表面漏电流 | $\propto n_i$ | $E_g/2$ |

**经验法则**：温度每升高 10°C，暗电流约翻倍。通过测量暗电流的激活能，可以判断主导机制：
- 激活能 ≈ $E_g$：扩散电流主导
- 激活能 ≈ $E_g/2$：产生-复合或表面漏电主导
- 激活能 ≈ 0：隧穿电流主导

### 暗电流对探测器性能的影响

#### 散粒噪声

暗电流产生的散粒噪声是探测器的主要噪声源之一：

$$i_{n,dark} = \sqrt{2qI_{dark}B}$$

其中 $B$ 为带宽。暗电流越大，噪声越大，可探测的最小光功率（NEP）越高。

#### 信噪比

$$SNR = \frac{I_{ph}^2}{2q(I_{ph} + I_{dark})B + i_{amp}^2}$$

弱光探测时 $I_{ph} \ll I_{dark}$，SNR 受暗电流限制。降低暗电流是提高探测器灵敏度的关键。

#### 降低暗电流的方法

1. **材料优化**：减少体材料缺陷和杂质浓度，提高 $\tau_{eff}$
2. **器件结构**：采用 PIN 结构扩大本征区、降低表面复合
3. **工艺改进**：优化台面钝化、减少表面态密度
4. **温度控制**：TEC 制冷可显著降低热激发暗电流
5. **材料选择**：在满足波长响应前提下，选择带隙较大的材料

## 总结

PN 结的定量分析建立在泊松方程和漂移扩散方程之上：
- **泊松方程**在耗尽近似下可解析求解耗尽区宽度和结电容
- **暗电流**由四种机制叠加：扩散电流（$\propto n_i^2$）、产生-复合电流（$\propto n_i$）、隧穿电流（电场相关）、表面漏电流（工艺相关）
- 通过激活能测量可以诊断暗电流的主导机制，指导器件优化方向

这些分析为理解光电探测器的灵敏度极限和 [光电器件与材料](10-optoelectronic-materials.md) 中的探测器性能参数提供了物理基础。
