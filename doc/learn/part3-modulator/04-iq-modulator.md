# IQ 调制器

> 正交幅度调制、星座图与高阶调制格式，现代光通信的核心技术

## 什么是 IQ 调制？

**IQ 调制（QAM 调制正交调制）**是一种将两个独立的基带信号（I 路和 Q 路）分别调制到相位相差 90° 的两个光载波上，然后合成一个信号的调制方式。IQ 调制可以同时控制光信号的幅度和相位，从而大大提高频谱效率。

$$E_{out}(t) = I(t) \cdot \cos(\omega t) + Q(t) \cdot \sin(\omega t)$$

其中 I 是同相 (in-phase) 分量，Q 是正交 (quadrature) 分量，两者相位差 90° (π/2)。利用三角函数的恒等式，上式可以改写为：

$$E_{out}(t) = A(t) \cdot \cos(\omega t + \phi(t))$$

其中幅度 A 和相位 φ 分别为：

$$A = \sqrt{I^2 + Q^2}, \quad \phi = -\arctan\left(\frac{Q}{I}\right)$$

## IQ 调制器的结构

一个典型的光 **IQ 调制器**由以下部分组成：

1. **光分束器**：将输入光分成两路，分别进入 I 臂和 Q 臂的 MZM。
2. **两个子 MZM（I-MZM 和 Q-MZM）**：分别对 I 路和 Q 路信号进行调制。每个子 MZM 工作在推挽模式，偏置在正交点以实现线性的电光转换。
3. **90° 光移相器（π/2 相移）**：在 Q 臂上引入一个额外的 π/2 相位偏移，使得 Q 路光与 I 路光正交。这个相移通常由偏置电压提供。
4. **光合束器**：将调制后的 I 路和 Q 路光合路输出。

### IQ 调制器结构框图

输入光 → 分束器 → I-MZM / Q-MZM + π/2相移 → 合束器 → 输出光

### 从 I-MZM / Q-MZM 到 IQ 信号的数学推导

设输入光场为 E_in，经分束器后等分为两路。I 路进入 I-MZM，Q 路经 π/2 相移后进入 Q-MZM。每个子 MZM 工作于推挽模式，分别由驱动电压 V_I 和 V_Q 控制。

#### I-MZM 输出（推挽模式，偏置在 Null 点）：

$$E_I = \frac{E_{in}}{2\sqrt{2}} \left[ \exp\left( +j\frac{\pi V_I}{V_\pi} \right) + \exp\left( -j\frac{\pi V_I}{V_\pi} \right) \right] = \frac{E_{in}}{\sqrt{2}} \cos\left( \frac{\pi V_I}{V_\pi} \right)$$

#### Q-MZM 输出（同样的偏置和驱动，附加 π/2 相移）：

$$E_Q = \frac{E_{in}}{2\sqrt{2}} \left[ \exp\left( +j\frac{\pi V_Q}{V_\pi} \right) + \exp\left( -j\frac{\pi V_Q}{V_\pi} \right) \right] \cdot e^{j\pi/2} = j \cdot \frac{E_{in}}{\sqrt{2}} \cos\left( \frac{\pi V_Q}{V_\pi} \right)$$

#### 合束后的总输出场：

$$E_{out} = E_I + E_Q = \frac{E_{in}}{\sqrt{2}} \left[ \cos\left( \frac{\pi V_I}{V_\pi} \right) + j \cos\left( \frac{\pi V_Q}{V_\pi} \right) \right]$$

当驱动电压在小信号范围（V << V_π）内变化时，cos 函数近似线性：cos(πV/V_π) ≈ 1 - (πV/V_π)²/2，交流耦合后可得 E_out ∝ V_I + j·V_Q，即 I 路控制实部、Q 路控制虚部，实现了复平面上的任意点调制。

## 星座图：直观理解 IQ 调制

**星座图 (Constellation Diagram)**是表示 IQ 调制信号的一种直观方式。它以 I 为横轴、Q 为纵轴建立复平面，每个符号对应星座图上的一个点，其位置由 I 和 Q 的值决定。

### IQ 信号与星座点的映射关系

#### QPSK 映射示例

QPSK 有 4 个星座点：

- 00 → I=-1, Q=-1 → 相位 225°
- 01 → I=+1, Q=-1 → 相位 315°
- 10 → I=-1, Q=+1 → 相位 135°
- 11 → I=+1, Q=+1 → 相位 45°

#### 16QAM 映射原理

16QAM 使用 4 个电平（±1, ±3）来表示 16 个星座点：

$$I, Q \in \{-3, -1, +1, +3\}$$

每个星座点的坐标 (I, Q) 由 2 bit 决定：
- 第 1-2 bit → I 值（00=-3, 01=-1, 11=+1, 10=+3）
- 第 3-4 bit → Q 值（同上）

驱动电压 V_I 和 V_Q 与 I/Q 值成正比。例如，对于 QPSK：V_I = +Vπ/2 或 -Vπ/2 分别对应 I = +1 或 -1。

### 格雷编码 (Gray Coding)

在星座图中，相邻星座点之间的比特映射通常采用**格雷编码**（Gray Coding）：任意两个相邻星座点（欧氏距离最小）的二进制码组仅相差 **1 bit**。

#### QPSK 格雷编码示例

- 00 (I=-1, Q=-1) ↔ 01 (I=+1, Q=-1) → 仅 Q 反转
- 00 (I=-1, Q=-1) ↔ 10 (I=-1, Q=+1) → 仅 I 反转
- 01 (I=+1, Q=-1) ↔ 11 (I=+1, Q=+1) → 仅 Q 反转

#### 与 BER 的关系

由于 AWGN 信道中最常见的错误是误判到相邻星座点，格雷编码确保每次符号错误只引起 **1 bit** 错误，而非多个比特同时错误。这使得 BER 与符号错误率 (SER) 的关系近似为：

$$\text{BER} \approx \frac{\text{SER}}{\log_2 M}$$

其中 M 是调制阶数。如果未采用格雷编码，BER 可能高出数倍。

### 星座图的信息：

- **幅度：**点到原点的距离 = √(I²+Q²)
- **相位：**点与横轴的夹角 = arctan(Q/I)
- **点数：**调制阶数（每个符号携带的比特数）

### 常见调制格式：

- **QPSK：**4 个星座点，每符号 2 bit
- **16QAM：**16 个星座点，每符号 4 bit
- **64QAM：**64 个星座点，每符号 6 bit

调制阶数越高，频谱效率越高，但对信噪比的要求也越高。在实际系统中，需要根据信道条件选择合适的调制格式。

## 调制格式与频谱效率

频谱效率 (Spectral Efficiency, SE) 是衡量调制技术优劣的重要指标，表示单位带宽内能够传输的数据速率，单位是 bit/s/Hz。

| 调制格式 | 星座点数 | 每符号比特 | 频谱效率 (bit/s/Hz) |
|----------|----------|------------|---------------------|
| OOK (开关键控) | 2 | 1 | ~1 |
| QPSK | 4 | 2 | ~2 |
| 16QAM | 16 | 4 | ~4 |
| 64QAM | 64 | 6 | ~6 |
| 256QAM | 256 | 8 | ~8 |

\* 实际频谱效率还与脉冲整形、编码开销等因素有关。

## IQ 调制器的偏置控制

IQ 调制器有三个需要精确控制的偏置点，它们对调制器的性能至关重要：

- **I-MZM 偏置**：控制 I 路子 MZM 的工作点，通常偏置在正交点（Null 点）以实现线性的强度-电压转换。
- **Q-MZM 偏置**：控制 Q 路子 MZM 的工作点，同样偏置在正交点。
- **正交相位偏置**：控制 I 路和 Q 路之间的相位差，确保为精确的 90° (π/2)。这个偏置的误差会导致 I/Q 不平衡，降低系统性能。

由于温度变化、老化等因素，偏置点会发生漂移。因此商用 IQ 调制器都配有自动偏置控制 (ABC) 电路，通过监测输出光的特性来动态调整偏置电压，维持最佳工作状态。

## IQ 不平衡及其影响

实际 IQ 调制器中，I 和 Q 两路可能存在幅度或相位的不平衡，这会导致星座图畸变，降低系统性能。

### IQ 不平衡的两种类型

#### 幅度不平衡 (Gain Imbalance)

I 路和 Q 路的增益不同，导致星座图在某一方向拉伸或压缩。

$$g = \frac{|I|}{|Q|} \neq 1$$

表现为星座图椭圆化，两个轴的半径不相等。

#### 相位不平衡 (Phase Imbalance)

I 路和 Q 路之间的相位差偏离 90°，导致星座图旋转或扭曲。

$$\theta \neq 90^\circ$$

表现为星座点偏离理想位置，形成菱形或不规则形状。

### IQ 不平衡的影响

- **镜像干扰：**IQ 不平衡会产生镜像信号，在解调时干扰原始信号。镜像抑制比 (IRR) 用于衡量这一影响：

  $$\text{IRR} = \frac{P_{image}}{P_{signal}}$$

- **误码率升高：**星座点偏离理想位置，判决边界不准确，导致误判概率增加。
- **频谱效率下降：**为补偿 IQ 不平衡，可能需要降低调制阶数或增加纠错码开销。

### IQ 不平衡补偿方法

- **发射端预补偿**：在驱动信号中预先加入反向不平衡，抵消调制器的非理想特性。需要精确测量调制器的不平衡参数。
- **接收端后补偿**：在 DSP 中使用自适应算法估计和补偿 IQ 不平衡。CMA/MMA 算法可以同时均衡信道和补偿 IQ 不平衡。

高性能 IQ 调制器通常将 IQ 不平衡控制在 < 1% 范围内，配合接收端 DSP 补偿，可以有效抑制其影响。

## 下一步：偏振复用

IQ 调制利用了光的幅度和相位维度，将频谱效率提升了数倍。但光还有一个重要的维度可以利用——**偏振**。

光有两个正交的偏振态（X 和 Y），它们可以作为独立的信道传输不同的数据。这种技术叫做**偏振复用 (PDM, Polarization Division Multiplexing)**，可以在不增加频谱宽度的前提下，使传输容量再翻一倍！

下一章我们将学习偏振的基本概念和偏振复用技术，然后将所有知识整合起来，认识现代高速光通信的核心——**高级调制 (Advanced Modulation)**。
