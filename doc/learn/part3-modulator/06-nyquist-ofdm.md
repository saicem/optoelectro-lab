# Nyquist 与 OFDM

脉冲整形与多载波调制技术

## DP-IQ 调制器回顾

**DP-IQ 调制器 (Dual-Polarization IQ Modulator)** 是高级调制技术的硬件基础。它将输入激光分为 X 和 Y 两个正交偏振态， 分别通过独立的 IQ 调制器进行调制，最后经偏振合束器 (PBC) 输出，实现了单波长 200G/400G 的传输速率。

### 结构组成
- 偏振分束器 (PBS)
- X 偏振 IQ 调制器 (I_x, Q_x)
- Y 偏振 IQ 调制器 (I_y, Q_y)
- 偏振旋转器 + 偏振合束器 (PBC)

### 传输容量
- DP-QPSK：100G (单波长)
- DP-16QAM：200G - 400G
- DP-64QAM：600G - 800G
- DP-256QAM：1T+

四路驱动信号 (I_x, Q_x, I_y, Q_y) 独立加载不同的数据，每个符号同时在幅度、相位和偏振三个维度上携带信息。 Nyquist 脉冲整形与光 OFDM 均可在这一平台上实现，进一步提升频谱效率。

## Nyquist 脉冲整形

在传统的 NRZ 调制中，矩形脉冲的频谱包含大量旁瓣，占用额外带宽。**Nyquist 脉冲整形** 通过**升余弦滤波器 (Raised-Cosine Filter)** 在发送端对脉冲进行整形， 将信号带宽严格限制在奈奎斯特频率以内。

$$H(f) = \begin{cases} T, & |f| \leq \frac{1-\alpha}{2T} \\ \frac{T}{2} \left[ 1 + \cos\left( \frac{\pi T}{\alpha} \left( |f| - \frac{1-\alpha}{2T} \right) \right) \right], & \frac{1-\alpha}{2T} < |f| \leq \frac{1+\alpha}{2T} \\ 0, & \text{otherwise} \end{cases}$$

### 升余弦滤波器时域波形示意

#### α = 0（理想 Nyquist）
sinc(t/T) 波形，在 t = ±T, ±2T... 处为零
带宽最小，但衰减慢

#### α = 0.1（常用）
升余弦波形，平滑过渡，快速衰减
频谱效率高，实用首选

#### α = 1（最大滚降）
最平滑波形，衰减最快
带宽翻倍，抗定时误差强

关键特性：在采样点 t = 0, T, 2T, 3T... 处，所有波形值都在零点， 因此相邻符号在该点不产生干扰 → 消除 ISI

### 滚降因子 α
- **α = 0**：理想 Nyquist，带宽 = 1/(2T)，不可实现
- **α = 0.1 ~ 0.2**：高速光通信常用，频谱效率高
- **α = 0.5 ~ 1**：带宽宽，但抗定时误差强

### Nyquist-WDM
对每个波长的信号进行 Nyquist 整形后，可以将波长间隔压缩到接近信号波特率，实现超高频谱效率的波分复用系统。

**核心优势**：消除码间干扰 (ISI) 的同时， 将频谱效率推向理论极限 2 baud/Hz， 是当前高速相干光通信系统的基石技术。

## 光 OFDM

**OFDM**（正交频分复用）是一种多载波调制技术，它将高速数据流分配到多个**正交子载波**上并行传输。 每个子载波上的符号速率较低，因此对色散和偏振模色散 (PMD) 有天然的容忍度。

### OFDM 子载波频谱示意

每个子载波的频谱峰值与其他子载波的零点位置重合 → 正交性保证无子载波间干扰

#### 正交条件
子载波间隔 Δf = 1/T_sym，每个子载波的频谱在其他子载波中心频率处恰好为零。 这一条件保证了子载波间的正交性，无需保护带。

#### 循环前缀 (CP)
将每个 OFDM 符号尾部的一部分复制到头部作为保护间隔，用于抵抗色散引起的符号间干扰。 CP 长度需大于信道最大时延扩展。

$$s(t) = \sum_{k=0}^{N-1} X_k \cdot e^{j2\pi k \Delta f t}, \quad 0 \leq t \leq T_{sym}$$

OFDM 信号可通过 IFFT 高效生成，接收端通过 FFT 恢复每个子载波上的调制符号。 这一计算效率是 OFDM 得以广泛应用的关键原因。

### CO-OFDM
- 使用相干接收，恢复完整的电场信息
- 频谱效率接近 Nyquist 单载波
- 可通过 FFT/IFFT 高效实现
- 无需复杂的时域均衡器

### 优劣势
- ✓ 对色散容忍度高
- ✓ 频谱利用率高
- ✗ 峰均功率比 (PAPR) 高
- ✗ 对光纤非线性敏感

## Nyquist vs OFDM 对比

Nyquist 脉冲整形单载波与光 OFDM 多载波是当前高速光通信中两种主流的先进调制方案。 两者在频谱效率上接近，但在实现复杂度、非线性容忍度和应用场景上各有千秋。

| 对比维度 | Nyquist 单载波 | 光 OFDM |
| --- | --- | --- |
| 调制解调 | 传统 DAC/ADC + 脉冲整形 | IFFT/FFT 实现 |
| 均衡复杂度 | 需要复杂时域均衡 | 每个子载波简单一抽头均衡 |
| PAPR | 低 | 高（主要缺点） |
| 非线性容忍度 | 较高 | 较低 |
| 色散容忍度 | 需要色散补偿 | 天生容忍 |
| DAC/ADC 要求 | 高带宽 | 带宽与子载波数相关 |
| 商用现状 | 主流方案 | 部分部署 / 研究 |

**现状**：商用高速相干光系统（100G/400G/800G）以 Nyquist 单载波为主流方案， 因其 PAPR 低、实现成熟。光 OFDM 则在长距离传输和弹性光网络中展现出独特优势， 两者在未来系统中将长期共存。

## 下一步：概率星座整形与编码

Nyquist 脉冲整形和光 OFDM 解决了"如何高效利用频谱"的问题。 然而，要逼近香农极限，还需要回答另一个关键问题：**"如何最优化地利用发射功率"**。

### 概率星座整形 (PCS)
通过不等概率的星座点分布，降低平均符号能量，获得成形增益。

### 前向纠错编码 (FEC)
通过 LDPC 等纠错码，在相同信噪比下获得更低的误码率。

PCS 与 FEC 的联合优化是当前 400G/800G 光通信系统的核心技术， 下一章将深入介绍这两项技术的原理、实现与工程应用。
