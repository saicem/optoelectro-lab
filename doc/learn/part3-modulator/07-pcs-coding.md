# 概率星座整形与编码

逼近香农极限的调制与纠错编码技术

## 概率星座整形 (PCS) 概念

**概率星座整形 (PCS)** 是近年来光通信领域最重要的突破之一。传统的 QAM 调制中，所有星座点等概率出现。 PCS 的核心思想是：**内部星座点出现概率更高，外部星座点出现概率更低**。

### 16QAM 星座点概率分布对比

#### 传统均匀分布
每个星座点概率 = 1/16 = 6.25%

#### PCS 不均匀分布
内部点概率更高（深色），外部点概率更低（浅色）

$$P(x_i) = \frac{e^{-\lambda |x_i|^2}}{\sum_j e^{-\lambda |x_j|^2}} \quad \text{(Maxwell-Boltzmann 分布)}$$

其中 λ 是整形参数（决定概率分布的陡峭程度），|x_i|² 是星座点到原点的距离平方（能量）。 λ 越大，内部点概率占比越高，成形增益越大，但信息速率损失也越大。 实际系统中 λ 根据信道条件动态调整，实现**速率自适应**。

### 速率自适应
- λ = 0 → 均匀分布，速率最大
- λ 增大 → 成形增益增加，速率降低
- 可灵活实现 0.125 bit/sym 的细粒度速率调整

### 与均匀 QAM 的对比
- 均匀 QAM 只能通过改变阶数（如 16→64）跳变速率
- PCS 可实现连续可调的传输速率
- 相同平均功率下，PCS 具有更低的误码率

## 成形增益的物理来源

为什么不等概率的星座点分布能够带来性能提升？ 这背后有着深刻的**信息论基础**。

### 成形增益的三个来源

1. **平均能量降低：** 内部星座点（低能量）出现概率高，外部星座点（高能量）出现概率低，使得平均符号能量降低，在相同发射功率下可获得更高的信噪比裕量。

2. **逼近球形星座：** 信息论证明，在加性高斯白噪声 (AWGN) 信道下，最优的输入分布是各向同性的高斯分布，其等概率轮廓面为球形。PCS 通过概率加权使均匀 QAM 星座的"平均形状"接近球形。

3. **香农极限：** 球形星座整形可以达到香农极限的理论增益上限 1.53 dB（二维），这是所有星座整形技术的理论最优值，来源于高斯分布相对于均匀分布的熵增益。

$$\text{成形增益} = \frac{E_{uniform}}{E_{PCS}} \leq 1.53 \text{ dB}$$

### 实际成形增益
- **1.53 dB**：二维理想成形增益上限（香农）
- **~0.8 dB**：PCS-16QAM 实际成形增益
- **~1.2 dB**：PCS-64QAM 实际成形增益
- **~1.4 dB**：PCS-256QAM 实际成形增益

### 维度的影响
成形增益随维度增加而增大。二维（2D）上限为 1.53 dB， 四维（4D，考虑偏振）上限为 ~1.76 dB， 无穷维度下可达 ~2.2 dB。实际光通信系统通常使用二维或四维整形。

**直观理解**：在星座图中，"靠近原点"的信号能量更低，但接收端更容易正确判决。 PCS 让发送端更多地选择"节能且可靠"的内部星座点，从而在整体上获得更好的传输性能。

## 前向纠错码 FEC 基础

**前向纠错码 (FEC)** 是光通信系统中不可或缺的组成部分。通过在发送端添加冗余校验位， FEC 使接收端能够检测并纠正传输过程中发生的错误，从而在相同信噪比 (SNR) 下实现更低的误码率。

### FEC 编码增益示意

#### 硬判决 FEC (HD-FEC)
- 对每个比特做 0/1 硬判决后再解码
- 实现简单，功耗低
- 编码增益约 5 - 6 dB
- 典型：RS(255,239)，BCH 码
- 100G 系统中广泛使用

#### 软判决 FEC (SD-FEC)
- 保留接收信号的幅度信息作为置信度
- 解码复杂度更高，性能更好
- 编码增益约 8 - 11 dB
- 典型：LDPC 码，Turbo 码
- 400G/800G 系统标配

### LDPC 码
LDPC (Low-Density Parity-Check) 码是一种线性分组码，其校验矩阵 H 中"1"的密度非常低（稀疏矩阵）。 这种稀疏性使得可以采用**置信传播 (Belief Propagation)** 算法进行高效迭代解码， 性能逼近香农极限。

$$H \cdot c^T = 0, \quad \text{其中 } H \text{ 是稀疏校验矩阵}$$

### 编码增益
- 定义为达到相同误码率所需的 SNR 降低量
- SD-FEC 编码增益通常比 HD-FEC 高 3 - 5 dB
- 编码增益随码长增加而增大

### 开销 (Overhead)
- HD-FEC：~7% 开销（如 RS(255,239)）
- SD-FEC：~15% - 27% 开销
- 开销越大 → 冗余越多 → 纠错能力越强 → 有效速率降低

## PCS 与 FEC 联合优化

PCS 和 FEC 并非独立工作，而是在实际系统中**联合优化**以逼近香农极限。 PCS 提供成形增益，FEC 提供编码增益，两者叠加的效果接近信息论的理论极限。

### 联合优化的核心架构

1. **CCDM 分布匹配器：** 恒参分布匹配器 (CCDM) 将均匀分布的输入比特转换为满足 Maxwell-Boltzmann 分布的概率符号序列， 再送入 FEC 编码器进行纠错编码。

2. **概率幅度整形 (PAS)：** 由 Bocherer 等人提出的 PAS 框架将 PCS 与 FEC 系统性地结合， 采用"恒参分布匹配 + 系统化 LDPC 编码"的方案，已成为 ITU-T 标准化的基础架构。

3. **互信息最大化：** 通过调整整形参数 λ 和 FEC 码率 R，使系统在给定 SNR 下达到最大的互信息，即最逼近香农极限的工作点。

### ITU-T 标准化
概率星座整形已被 ITU-T G.698.2 采纳为光通信系统的推荐技术。 主流光模块厂商（如华为、Ciena、Infinera）已在 400G/800G 产品中部署 PCS + LDPC 联合方案， 实现了接近香农极限 0.5 dB 以内的传输性能。

### PCS + SD-FEC 实际效果
- 相比传统均匀 QAM + HD-FEC 提升约 3 - 4 dB
- 400G 传输距离延长 30% - 50%
- 支持 0.125 bit/sym 粒度的速率自适应

### 实现挑战
- DSP 芯片功耗和面积增加
- PAS 框架引入的速率损失
- CCDM 匹配器的硬件实现开销

## 综合对比与展望

回顾整个 Part 3，我们从干涉原理出发，经历了 MZ 调制器、IQ 调制器、偏振复用，再到 Nyquist 脉冲整形、 光 OFDM，最后到概率星座整形与 FEC 编码。 每一项技术都在推动光通信系统**逼近香农极限**。

| 技术 | 核心贡献 | 增益类型 | 典型增益 |
| --- | --- | --- | --- |
| Nyquist 整形 | 压缩带宽，消除 ISI | 频谱效率 | ≤ 2 baud/Hz |
| 光 OFDM | 抗色散，均衡简单 | 链路预算 | 取决于色散 |
| 概率星座整形 | 降低平均能量 | 成形增益 | ≤ 1.53 dB |
| SD-FEC (LDPC) | 纠错编码 | 编码增益 | ≤ 11 dB |
| PCS + FEC | 联合优化 | 综合增益 | 距香农极限 < 0.5 dB |

### 未来方向

#### 空分复用 (SDM)
多芯光纤 (MCF) 和少模光纤 (FMF) 利用空间维度进一步提升传输容量， 是超越单模光纤香农极限的关键技术。

#### 智能编码调制
基于机器学习的端到端光通信系统优化，自动学习最优的星座形状和编码方案， 进一步逼近香农极限。

#### 多级编码调制
将不同可靠性的比特分配给不同保护级别的编码，结合 PCS 和概率整形， 实现更细粒度的速率自适应。
