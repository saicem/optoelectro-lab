# 光波基础与物理量

从电磁波谱出发，理解光波的基本物理量、数学描述与折射率概念

## 电磁波谱与光通信波段

光是电磁波谱中的一部分。整个电磁波谱按频率从低到高（波长从长到短）排列，包括无线电波、微波、红外光、可见光、紫外光、X 射线和 γ 射线。 光通信使用的波长集中在近红外区域（~800–1700 nm）。

在近红外区域，光通信根据波长划分了多个标准波段。 其中 C 波段（Conventional Band）因为处于石英光纤的最低损耗窗口，成为长距离光通信的主力波段。

### 光通信波段速览

- **O-band** — Original — 零色散窗口，最早用于光通信（1260–1360 nm）
- **E-band** — Extended — 水吸收峰高，损耗较大（1360–1460 nm）
- **S-band** — Short wavelength — 短波长方向扩展（1460–1530 nm）
- **C-band** — Conventional — 最低损耗 (~0.2 dB/km)，长距离骨干网（1530–1565 nm）
- **L-band** — Long wavelength — 长波长扩展，用于 DWDM 扩容（1565–1625 nm）

## 光波的基本物理量

光是一种电磁波，在真空中的传播速度为恒定值 c ≈ 3×10⁸ m/s。 波长、频率与光速满足基本关系：

$$c = \lambda \cdot f$$

其中 λ 为波长，f 为频率。 光子的能量由普朗克关系给出：

$$E = hf = \frac{hc}{\lambda}$$

h ≈ 6.626×10⁻³⁴ J·s 为普朗克常数。 频率越高（波长越短），光子携带的能量越大。在 C 波段标准波长 1550 nm 处， 光子能量约为 0.8 eV。

- 可见光波长范围：~380–780 nm
- 可见光频率范围：~430–790 THz
- 光子能量范围：~1.6–3.3 eV

### 电场与调制的关系

电场强度 E 描述了空间中某点所受电场力的大小和方向。在均匀电场中，电场强度与电压和距离的关系为：

$$E = \frac{V}{d}$$

其中 V 为外加电压，d 为电极间距。 在光调制器中，外加电压在电极之间产生电场，通过电光效应改变材料的折射率，从而实现对光波的相位调制——这是几乎所有高速光调制器的核心原理。

## 简谐光波的数学描述

单色平面波可以用简谐函数来描述。沿 z 方向传播的光波，其电场随空间和时间的完整表达式为：

$$E(z,t) = E_0 \cos(\omega t - kz + \phi_0)$$

### 各参数含义

- **E₀** — 电场振幅，决定光强
- **ω = 2πf** — 角频率，单位 rad/s
- **k = 2π/λ** — 波数，描述空间周期性
- **φ₀** — 初相位，决定 t=0 时刻的振动状态

### 光强与振幅

光的强度（能流密度）与振幅的平方成正比：

$$I \propto |E_0|^2$$

这一关系是强度调制的物理基础——通过改变振幅即可控制光强， 从而将信息编码到光载波上。

## 相位与相位差

相位是波动的核心概念，它描述了振动在某一时刻所处的状态——是波峰、波谷还是中间某处。 两束光之间的相位差决定了它们相遇时是相互加强还是相互削弱，这正是干涉的本质。

$$\Delta\phi = \phi_1 - \phi_2 = \frac{2\pi}{\lambda} \cdot \Delta L$$

其中 ΔL 是两束光的光程差。 当光程差为波长的整数倍时，相位差为 2π 的整数倍，两束光同相，发生相长干涉； 当光程差为半波长的奇数倍时，相位差为 π 的奇数倍，两束光反相，发生相消干涉。

### 相长干涉（加强）

相位差 Δφ = 2kπ (k 为整数)
光程差 ΔL = kλ

### 相消干涉（削弱）

相位差 Δφ = (2k+1)π
光程差 ΔL = (2k+1)λ/2

马赫-曾德调制器正是利用这一原理：通过电光效应改变其中一臂的折射率，引入受控的相位差，从而将相位调制转换为强度调制。

## 折射率：光与物质的相互作用

折射率是描述光在介质中传播速度变化的基本物理量，它架起了光和物质相互作用的桥梁。

### 折射率的定义

$$n = \frac{c}{v}$$

其中 n 为折射率，c 为真空光速，v 为光在介质中的传播速度。 由于 v ≤ c， 所有介质的折射率均 n ≥ 1。

- 空气 (标准状态)：≈ 1.0003
- 水：≈ 1.33
- 石英玻璃 (SiO₂)：≈ 1.44
- 铌酸锂 (LiNbO₃)：≈ 2.2

### 折射率在光调制中的核心作用

折射率并非固定不变——它可以通过外加电场来改变，这就是电光效应。 对于某些晶体（如铌酸锂），折射率的变化与外加电场呈线性关系（普克尔效应）：

$$\Delta n = -\frac{1}{2} n^3 r E$$

其中 n 是材料的折射率，r 是电光系数（材料固有属性），E 是外加电场强度。 折射率的微小变化会导致光在介质中传播时积累显著的相位变化：

$$\Delta\phi = \frac{2\pi}{\lambda} \cdot \Delta n \cdot L$$

光调制器正是通过外加电信号控制折射率、进而控制光相位， 最终实现将电信号"写入"光载波的目标。

### 折射率 vs. 波长（色散）

折射率还随光波长变化而变化，这一现象称为色散。 在光纤中，不同波长的光传播速度不同，导致脉冲展宽。 这是光通信系统设计中的重要考虑因素。

## 从基础到调制

在光通信系统中，我们需要将电信号加载到光载波上进行传输， 这个过程叫做光调制。 理解了光波的基本物理量、相位和折射率概念之后，现在就来看看我们可以利用光的哪些属性来编码信息。

### 光调制的六大维度

1. **强度（幅度）** — 控制光的亮/暗来表示 0/1。直接调制和强度调制器都属于这类。
2. **相位** — 通过 I/Q 两路信号协同控制，实现幅度和相位的联合调制，大幅提升频谱效率。
3. **偏振** — 利用光的两个正交偏振态作为独立信道，使容量再翻一倍。
4. **频率 / 波长** — 不同波长承载不同数据流。WDM 波分复用是使光纤容量从 Gb/s 跃升至 Tb/s 的关键技术。
5. **时间** — 将时间划分为多个时隙，每个时隙传输不同的信号。光时分复用 (OTDM) 可将单波长速率推向 Tb/s。
6. **空间** — 多芯光纤 / 少模光纤 (SDM) 通过空间维度并行传输，是突破香农极限的终极手段。

### 1. 直接调制 vs 外部调制

直接调制通过改变激光器的注入电流来调制光强，简单但速率受限、会产生啁啾。 外部调制器则对连续输出的激光进行调制，可实现更高的调制速率和更好的性能。

### 2. 电光效应的工程实现

基于电光效应的外部调制器是高速光通信的核心器件。 通过将电光晶体（如铌酸锂）集成在干涉仪结构中，电信号的变化被精确地转换为光强度的变化——这就是马赫-曾德调制器的基本原理， 后续章节将详细讲解。

### 3. 调制技术的发展路径

从简单的强度调制（OOK），到复杂的 IQ 正交调制（QPSK/QAM），再到偏振复用（PDM），光调制技术不断进步，使得单根光纤的传输容量从 Gb/s 量级提升到 Tb/s 甚至 Pb/s 量级。

OOK → QPSK → 16QAM → PDM-16QAM
